# 🎮 Hub de Mini-Jeux Unity 🎮

Ce projet est une collection de mini-jeux interactifs développés sous Unity. L'objectif est de proposer des expériences rapides, compétitives et fun pour deux joueurs ou plus.

## 🚀 Jeux Actuels
1.  **Le Pendu de Bob (Mode Duel)** :
    * Un duel de lettres avec des vies séparées.
    * Système de bonus "Bob" pour protéger le perdant temporairement.

## 🛠 Prochainement (Roadmap)

## 📁 Organisation du Projet
Pour garder le projet propre, nous utilisons la structure suivante dans le dossier `Assets/` :
* `_Common/` : Scripts et sons partagés par tous les jeux.
* `Pendu/` : Scènes, scripts et visuels du Pendu.
* `Reflexe/` : Futur dossier pour le prochain jeu.

## 📲 Exportation (Android)
Les versions de test sont disponibles dans le dossier `AAA Builds/Android/`. 
Format : `.apk`
